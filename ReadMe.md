# 📚 SmartLib  
### A Web-Based Interactive Platform for Streamlined Book Search, Room Reservation, and Borrower Management in Caraga State University

---

## 📌 Overview

**SmartLib** is a modern, web-based interactive library management system designed for **Caraga State University (CSU)**.  

The current CSU Library website lacks essential digital services such as online book searches, room reservations, and automated borrower tracking. SmartLib aims to redesign and upgrade the platform into a fully interactive and responsive system that enhances accessibility, efficiency, and overall user experience.

The system enables users to:

- 🔎 Search books and theses online  
- 📖 Track borrowing and in-house reading activities  
- 🏢 Reserve discussion rooms in real time  
- 📱 Access services across desktop, tablet, and mobile devices  

---

## 🚨 Problem Statement

The existing CSU Library system faces several challenges:

- ❌ No online search functionality for books and theses  
- ❌ Manual borrowing transactions (error-prone and time-consuming)  
- ❌ No real-time tracking of in-house reading activities  
- ❌ No online room reservation system  
- ❌ Overlapping room bookings due to lack of scheduling automation  
- ❌ Outdated, non-responsive website design  

These limitations create operational inefficiencies for both students and librarians.

---

## 🎯 Project Objectives

SmartLib aims to:

1. **Redesign the CSU Library website** to be fully responsive and interactive.
2. **Implement a digital borrowing and tracking system** to replace manual logbooks.
3. **Develop an integrated online search system** with advanced filtering:
   - Title
   - Author
   - Category
4. **Create a real-time discussion room reservation system**.
5. **Monitor in-house reading activities** using a centralized database.
